Instructions for use						Last Update : 2020/09/17


System Environment : Ubuntu16.04  


Steps to use :

 	1. Copy bash file to the root directory. 
	2. Run sudo bash opencv_install.sh
	3. After running it, go to https://github.com/AlexeyAB/darknet to download Darknet.
           And install Yolov4 according to the Github document.
